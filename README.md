# Isotope ⏱️

A sleek macOS menubar timer app with stopwatch, countdown timer, and Pomodoro modes.

![macOS](https://img.shields.io/badge/macOS-12.0+-blue)
![Swift](https://img.shields.io/badge/Swift-5.0-orange)

## Features

- **Stopwatch** - Count up from 00:00
- **Timer** - Countdown with presets (5-60 min) or custom duration
- **Pomodoro** - 25 min work → 5 min break, long break every 4 sessions
- **Compact Mode** - Vertically stacked display for minimal menubar space
- **10 Gradient Themes** - Sunset, Ocean, Forest, Neon, Fire, Mint, Gold, Lavender, Rainbow, Aurora
- **9 Solid Colors** - Customize your timer display

## Installation

### Download DMG
Download the latest release from [Releases](https://github.com/YOUR_USERNAME/Isotope/releases).

### Build from Source
1. Clone the repository
2. Open `Isotope.xcodeproj` in Xcode
3. Press `⌘R` to build and run

## Keyboard Shortcuts

| Action | Shortcut |
|--------|----------|
| Start/Stop | `⌘S` |
| Reset | `⌘R` |
| Skip Phase (Pomodoro) | `⌘K` |
| Quit | `⌘Q` |
